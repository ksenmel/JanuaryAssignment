# JanuaryAssignment

## Задание

Задан ориентированный граф в виде списка смежности. Напишите программу, определяющую, есть ли в графе циклы

## Описание решения
Алгоритм ```has_cycle``` в [src/cycles.py](https://github.com/ksenmel/JanuaryAssignment/blob/main/src/cycles.py) проверяет наличие цикла в направленном графе с помощью поиска в глубину (DFS)

В алгортме используется два множества для отслеживания состояния вершин:
- ```in_progress```: вершины, которые находятся в процессе обработки
- ```visited```: вершины, которые полностью обработаны

Более подробная информация о входных данных и возвращаемых значениях находится в документациив [src/cycles.py](https://github.com/ksenmel/JanuaryAssignment/blob/main/src/cycles.py)
